## 커밋 메시지 컨벤션

### 📌 기본 형태
```
<타입>: <메시지>

예)
feat: 로그인 UI 추가
fix: 캘린더 날짜 오류 수정
refactor: 일정 정렬 로직 개선
```

### 📌 타입 목록
| 타입 | 설명 |
|------|------|
| feat | 새로운 기능 추가 |
| fix | 버그 수정 |
| docs | 문서 추가/수정 |
| style | 코드 스타일 수정 (포맷팅, 세미콜론 등) |
| design | 사용자 UI 디자인 변경 |
| refactor | 리팩토링, 구조 개선 |
| test | 테스트 코드 추가/수정 |
| chore | 빌드/환경/기타 작업 |
| rename | 파일 혹은 폴더 명을 수정만 한 경우 |
| remove | 파일을 삭제만 한 경우 |
